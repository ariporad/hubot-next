# hubot-next

Another Satisfied Customer

See [`src/next.coffee`](src/next.coffee) for full documentation.

## Installation

In hubot project repo, run:

`npm install hubot-next --save`

Then add **hubot-next** to your `external-scripts.json`:

```json
[
  "hubot-next"
]
```

## Sample Interaction

```
user1>> hubot hello
hubot>> hello!
```
